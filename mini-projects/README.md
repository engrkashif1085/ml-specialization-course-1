# Mini Projects

This folder contains small machine learning mini-projects based on **Multiple Linear Regression** and **Classification** concepts learned in **Coursera Machine Learning Specialization Course 1: Supervised Machine Learning - Regression and Classification**.

The purpose of these mini-projects is to apply theory into practical examples and improve hands-on machine learning skills.

## Project Goals

The main goals of this folder are:

* Practice Multiple Linear Regression concepts
* Practice Classification concepts
* Work with multiple input features
* Understand the complete machine learning workflow
* Improve Python, NumPy, Pandas, Matplotlib, and Scikit-learn skills
* Build small regression and classification projects for learning and portfolio improvement

## Suggested Mini Projects

### 1. House Price Prediction

Use multiple linear regression to predict house prices based on different features such as house size, number of bedrooms, number of bathrooms, location-related values, and age of the house.

Main concepts:

* Multiple Linear Regression
* Feature Selection
* Feature Scaling
* Cost Function
* Gradient Descent
* Model Evaluation

### 2. Car Price Prediction

Use multiple linear regression to predict car prices based on features such as car model year, mileage, engine size, fuel type, transmission type, and brand.

Main concepts:

* Multiple Linear Regression
* Data Preprocessing
* Handling Categorical Features
* Feature Scaling
* Model Training
* Prediction

### 3. Medical Insurance Cost Prediction

Use multiple linear regression to predict medical insurance costs based on features such as age, BMI, number of children, smoking status, gender, and region.

Main concepts:

* Multiple Linear Regression
* Data Cleaning
* Feature Engineering
* Handling Categorical Data
* Model Evaluation
* Prediction

### 4. Diabetes Prediction

Use classification to predict whether a person is likely to have diabetes or not based on health-related features such as glucose level, BMI, age, blood pressure, insulin level, and other medical measurements.

Main concepts:

* Classification
* Logistic Regression
* Sigmoid Function
* Decision Boundary
* Data Preprocessing
* Model Evaluation
* Accuracy, Precision, and Recall

## Basic Machine Learning Workflow

Each mini-project should follow this simple workflow:

1. Define the problem
2. Load the dataset
3. Explore and understand the data
4. Clean or prepare the data
5. Select useful features
6. Train the machine learning model
7. Make predictions
8. Evaluate the model
9. Write key observations
