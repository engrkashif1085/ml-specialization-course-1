## Key Concepts

### Machine Learning

Machine learning is a field of artificial intelligence where computers learn patterns from data and make predictions or decisions without being directly programmed for every rule.

### Supervised Learning

In supervised learning, the model learns from labeled data. This means the input data already has correct output values.

Examples:

* Predicting house prices
* Predicting exam scores
* Classifying emails as spam or not spam

### Linear Regression

Linear regression is used to predict continuous values. It tries to find the best straight line that fits the data.

### Cost Function

The cost function measures how wrong the model predictions are. A lower cost means the model is performing better.

### Gradient Descent

Gradient descent is an optimization algorithm used to reduce the cost function and improve the model step by step.
