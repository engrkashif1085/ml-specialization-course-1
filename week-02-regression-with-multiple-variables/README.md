## Key Concepts

### Multiple Linear Regression

Multiple linear regression is used when we have more than one input feature. For example, predicting house price using size, number of bedrooms, location, and age of the house.

### Vectorization

Vectorization helps make machine learning code faster and cleaner by using NumPy arrays instead of writing many loops manually.

### Feature Scaling

Feature scaling adjusts the range of input features so that gradient descent can work faster and more smoothly.

### Learning Rate

The learning rate controls how big each step of gradient descent should be. A very small learning rate can make training slow, while a very large learning rate can make training unstable.

### Feature Engineering

Feature engineering means creating new useful input features from existing data to improve model performance.

### Polynomial Regression

Polynomial regression is used when the data does not fit well with a straight line. It allows the model to learn curved patterns.
